---

I know, I know. Rust this, rust that. Rust is a low level programming language and people are praising it. I get it. It sound stupid, but I'm switching to Rust. (With Go and Java on the side.)

## Why have I made the switch, exactly?
Well, recently, I've been learning about Rust and clearly impressed by it's performance and safety features. Compiling did take long, but it was worth it. `4mb` for a CLI tool that I made? Hell yeah! Check out the CLI tool here; https://github.com/daveberrys/LitV

## Will I be going back to Python?
Depends on the situation. If it's a simple script or tool, I'm obviously going to use python for that specific project. But if it's complex? That's where I use Rust.

## What about Java or Go?
Probably random side projects. I'm currently learning Java because of, well, school. Learning Go because it's interesting.
